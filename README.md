# lol-cli

Basic Ruby CLI with Thor. Adapted from http://willschenk.com/making-a-command-line-utility-with-gems-and-thor/

## Installation

    $ rake install

## Usage

    $ lol version
    $ lol greeter hello John
    $ lol greeter hello John --upcase
